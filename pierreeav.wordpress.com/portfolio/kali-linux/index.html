Kali Linux and tools

When I start wanted to go deeper in cybersecurity, one of the first thing I have done is to install a Kali Linux virtual machine, to master Linux shell and to familiarize with all the specialize tools that Kali can bring to you.

So after I installed my VM, I started playing with this distribution, to realize some challenges on it, and I worked and I learned to manage the main softwares available on Kali, like BurpSuite, Nmap, Metasploit, Wireshark, John, Hydra, Hashcat…

But cybersecurity is in constant evolution, and I learned new things and tricks everyday!
